Image Sizes for Assets
======================
Each asset has its own folder with 5 images (1 card + 4 gallery).

Image specs:
- card.png: 1600x1000 (16:10 landscape) - for asset grid cards & project resource cards
- 1.png to 4.png: 1414x2000 (~2:3 portrait) - for asset detail page gallery

Folder structure:
src/images/Assets/
├── {asset-slug}/
│   ├── card.png (16:10 landscape - for cards)
│   ├── 1.png (portrait - gallery)
│   ├── 2.png (portrait - gallery)
│   ├── 3.png (portrait - gallery)
│   └── 4.png (portrait - gallery)
