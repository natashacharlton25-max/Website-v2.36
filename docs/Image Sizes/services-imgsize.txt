SERVICES PAGE IMAGES
====================
Location: src/images/Services/
Used by: src/pages/services.astro (Offerings section)

NAMING CONVENTION
-----------------
Format: offering-[number]-[short-name].png
Examples:
  offering-1-resources.png     → Ready-to-Use Resources
  offering-2-whitelabel.png    → White-Label Packages
  offering-3-custom.png        → Custom Resource Development
  offering-4-training.png      → Training and Licensing

IMAGE SPECIFICATIONS
--------------------
Recommended size: 800x600px (4:3 aspect ratio)
Format: PNG
Astro will optimize automatically
