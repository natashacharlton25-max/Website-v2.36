AUTHORS IMAGES
==============
Location: src/content/authors/[author-slug]/
Used by: src/pages/insights/[slug].astro (About the Author section)

FOLDER STRUCTURE
----------------
src/content/authors/
└── natasha-charlton/
    ├── index.md        (author data)
    └── avatar.png      (author photo)

NAMING CONVENTION
-----------------
Folder: [firstname-lastname] in lowercase with hyphens
Image: avatar.png (consistent across all authors)

ADDING A NEW AUTHOR
-------------------
1. Create folder: src/content/authors/firstname-lastname/
2. Add avatar.png (author photo)
3. Create index.md with frontmatter:
   ---
   name: "Full Name"
   role: "Role Title"        # optional
   bio: "Short bio text..."
   photo: "./avatar.png"
   credentials: []           # optional
   specialties: []           # optional
   website: ""               # optional
   ---

IMAGE SPECIFICATIONS
--------------------
Filename: avatar.png
Recommended size: 400x400px (1:1 square)
Format: PNG (for transparency support)
Usage: Displayed at 80-120px with rounded borders

CURRENT AUTHORS
---------------
natasha-charlton/avatar.png → Natasha Charlton (Founder)

USAGE IN INSIGHTS
-----------------
In insight frontmatter, set author to the folder name:
  author: "natasha-charlton"

The insights/[slug].astro page automatically loads the author
data and displays it in the "About the Author" section.
